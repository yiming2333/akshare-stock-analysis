## 项目简介

本工具基于 **AKShare + Baostock + pytdx + 新浪财经** 多数据源，对 A 股进行 **四维深度诊断**：

- **基本面**：股本、净资产、ROE、利润增速、现金流质量、PE/PB 历史百分位、同行对比、DCF 估值
- **技术面**：MA、MACD、KDJ、RSI、WR、BOLL、ADX、ATR、量价关系、多周期趋势
- **资金面**：主力/超大单/大单/中单/小单净流入、北向资金、融资融券、股东户数
- **消息面**：东财新闻、研报评级、解禁预警

**v3.0 新增**：
- **代码架构重构**：800行巨型函数拆分为 20+ 独立模块，可维护性质变
- **Bug 修复**：6 项已知 Bug 全部修复（详见版本历史）
- **新增 RSI 回测策略**：`--rules=rsi_oversold` 超卖反弹策略
- **日志系统**：替代静默异常吞没，问题可追踪
- **输入校验**：股票代码格式验证
- **动态日期**：K线/融资/股东数据不再硬编码截止日期
- **--quick 模式真正生效**：跳过新闻/北向/融资/股东/研报/解禁，减少 API 调用

## 安装与依赖

### 环境要求
- Python 3.9+（推荐 3.11）
- Windows / Linux / macOS

### 安装步骤

1. **克隆仓库**（或直接下载脚本）
```bash
git clone https://github.com/yiming2333/akshare-stock-analysis.git
cd akshare-stock-analysis
```

2. **安装依赖**
```bash
pip install baostock pytdx akshare pandas numpy requests
```

> 如果遇到网络问题，可使用国内镜像源：`pip install -i https://pypi.tuna.tsinghua.edu.cn/simple baostock pytdx akshare pandas numpy requests`

3. **验证安装**
```bash
python scripts/akshare_query.py 600519 --quick
```
（首次运行会自动创建缓存目录 `.cache` 和 SQLite 数据库 `snapshots.db`）

## 使用方法

### 基础诊断
```bash
python scripts/akshare_query.py 600519
```
输出完整的 JSON 诊断报告（包含所有 sections）。

### 快速模式（跳过新闻/北向/融资/股东/研报/解禁，减少 API 调用）
```bash
python scripts/akshare_query.py 600519 --quick
```

### 保存结果到文件
```bash
python scripts/akshare_query.py 600519 --output report.json
```

### 回测模式
```bash
# MACD 金叉策略（默认）
python scripts/akshare_query.py 600519 --backtest

# RSI 超卖反弹策略
python scripts/akshare_query.py 600519 --backtest --rules=rsi_oversold
```
输出：总交易次数、胜率、累计收益率、最大回撤、夏普比率、详细交易记录。

### 趋势追踪（使用 SQLite 持久化，输出 20 日趋势序列）
```bash
python scripts/akshare_query.py 600519 --trend
```
会在 JSON 中增加 `trend_history` 和 `trend_summary` 字段。

### 批量分析（CSV 文件，首列为股票代码）
```bash
python scripts/akshare_query.py --batch stocks.csv
```
示例 `stocks.csv`：
```csv
stock_code
600519
000858
300750
```
输出汇总对比表（含名称、收盘价、PE、评分、信号数、风险数、估值状态）。

### 跳过特定数据源
```bash
python scripts/akshare_query.py 600519 --no-fund-flow   # 跳过资金流
python scripts/akshare_query.py 600519 --no-news        # 跳过新闻
```

| `technical` | MA系统、MACD、KDJ、RSI、WR、BOLL、ADX、ATR、量价关系 |
| `trends` | 5/10/20/30/60/120日涨跌幅及高低点 |
| `volume_price` | 当日量比、5日量价序列 |
| `fund_flow` | 最近10日主力/超大单/大单/中单/小单净流入、今日数据、5日总计、背离预警 |
| `northbound` | 北向资金近20日持股变化、5日/20日净额、趋势判断 |
| `margin` | 融资融券余额、占流通市值比、融资买入/偿还 |
| `shareholders` | 最新股东户数、变化、筹码集中趋势 |
| `analyst` | 研报列表、买入占比、评级变动统计 |
| `lockup` | 未来60天解禁明细、预警等级 |
| `risk_metrics` | 20日/60日年化波动率、95% VaR、1年最大回撤及风险评级 |
| `signals` | **客观信号数组**（技术/资金/北向/基本面信号） |

## 技术指标详解

| 指标 | 参数 | 计算方法 |
|:---|:---|:---|
| **MA** | 5/10/20/30/60/120 | 简单移动平均 |
| **MACD** | (6,13,5) | DIF = EMA(6)-EMA(13), DEA = EMA(DIF,5), HIST = 2×(DIF-DEA) |
| **KDJ** | (6,3,3) | RSV = (C-L6)/(H6-L6)×100, K=EMA(RSV,2), D=EMA(K,2), J=3K-2D |
| **RSI** | 6/12/24 | RS = 平均涨幅/平均跌幅, RSI = 100 - 100/(1+RS) |
| **WR** | 10 | (H10-C)/(H10-L10)×100 |
| **BOLL** | (20,2) | MID=MA20, UP=MID+2×STD, DOWN=MID-2×STD |
| **ADX** | 14 | +DI/-DI 计算后取 DX 平滑 |
| **ATR** | 14 | 真实波动幅度(TR)的14日平均 |
| **量比** | 5日 | 当日成交量 / 5日均量 |

## 评分与信号体系

### 客观信号 (`signals`)
脚本自动识别并输出以下信号（无主观建议）：

**技术信号**
- MACD 金叉/多头运行/空头
- KDJ 金叉/超买
- 均线多头排列(5条以上)/空头排列
- RSI 超买(>80) / 超卖(<20)
- BOLL 上轨突破 / 下轨超卖

**资金信号**
- 主力连续5日净流入/流出（含总额）
- 聪明钱底背离（主力买+股价跌）

**北向信号**
- 5日大幅流入(>5亿) / 大幅流出(<-5亿)

**基本面信号**
- ROE 优秀 (>15%)
- 净利润高增长 (>50%) / 较快增长 (>20%)
- PEG 深度低估 (<0.5) / 偏低 (<1.0)
- PE 历史低位 (<20%分位) / 极高位 (>90%分位)

**估值信号**
- 相对估值高估/低估（基于同行PE/PB百分位）

### 风险预警 (`risk_warnings`)
- **解禁**：未来60天大额解禁(>10%)或中等解禁(>5%)
- **融资过热**：融资余额占流通市值 >8%
- **估值峰值**：PE历史分位 >90%
- **PEG高估**：PEG >3
- **现金流风险**：经营现金流为负
- **筹码分散**：股东户数增加 >5%
- **高Beta**：Beta >1.5
- **高回撤**：1年最大回撤 < -30%

### 综合评分规则（用于 `assessment.score`）

| 维度 | 加分项 | 减分项 |
|:---|:---|:---|
| **均线** | 多头排列 ≥5条 +1 | — |
| **MACD** | DIF>DEA +1 | — |
| **KDJ** | K>D +1 | J>100 -1 |
| **RSI** | 30~70之间 +1 | 超买或超卖 +1警告 |
| **WR** | — | <20 -1 (警告) |
| **BOLL** | — | 位置>80% -1 (警告) |
| **ADX** | 强趋势且方向为多头 +1 | 强趋势但方向为空头 -1 |
| **ROE** | >8% +1 | — |
| **净利润增速** | >10% +1; >50% +2 | — |
| **现金流质量** | strong +1 | weak -1 |
| **PEG** | <0.5 +1 | >2 -1; >3 -2 |
| **PE历史分位** | <20% +1 | ≥80% -1; ≥90% -2 |
| **同行排名** | 前2名 +1 | 倒数第2名 -1 |
| **主力资金** | 净流入 +1 | 股价涨但主力卖 -2 |
| **北向资金** | 5日净额>1亿 +1; >5亿 +2 | 5日净额<-1亿 -1; <-5亿 -2 |
| **融资杠杆** | 低杠杆 +1 | 高杠杆 -1+警告 |
| **股东户数** | 集中 +1 | 分散 -1 |
| **研报买入占比** | ≥60% +1; ≥80% +2 | <20% -2 |
| **解禁** | — | >5% -1; >10% -2+警告 |
| **舆情情感** | 正面词计分净额 ≥2 +1~2 | 负面词计分净额 ≥2 -1~2 |

> 最终得分不构成投资建议，仅反映多空证据强度。

## 注意事项

1. **数据源稳定性**：AKShare、Baostock、pytdx 均为免费数据接口，可能因源网站变化导致接口失效。**建议锁定依赖版本**，并配合缓存和重试机制使用。
2. **非交易时段**：实时行情（pytdx/新浪）在休市时可能返回旧数据，资金流向和北向数据当日无更新。
3. **频率限制**：避免在短时间内多次批量查询，以免被数据源限流。`--batch` 模式下每个股票依次分析，自动延迟 1 秒。
4. **投资用途**：本工具**仅限个人学习和研究**，所有输出（包括信号、估值、预测）均基于公开数据客观计算，**不构成任何买入/卖出建议**。使用者应自行承担风险。
5. **DCF 简易模型**：采用固定 WACC=10%、永续增长率=3%，三场景增长假设（保守5%/基准10%/乐观20%），仅作为粗略参考，实际价值需结合行业与公司具体情况调整。
6. **回测局限性**：回测策略为固定规则（MACD金叉买入、持5日卖出、-8%止损；或RSI超卖买入、持3日卖出、-5%止损），不代表未来表现，且未考虑交易成本、滑点。
